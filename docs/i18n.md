# TimeTask 多语言说明

## 已实现能力
- 运行时语言切换（简体中文 / English）。
- 语言选择持久化到用户设置（`UiLanguage`）。
- 主界面核心文案、设置菜单、导入/导出/关于等关键提示的双语资源化。

## 资源文件
- 默认语言（简体中文）：`Properties/Resources.resx`
- 英文：`Properties/Resources.en-US.resx`

## 在 XAML 中使用
```xaml
Text="{local:Loc Key=Window_Main_Title}"
```

## 在 C# 中使用
```csharp
I18n.T("Title_Error")
I18n.Tf("Message_OperationFailedFormat", ex.Message)
```

## 新增翻译条目流程
1. 在 `Properties/Resources.resx` 增加 key 与中文值。
2. 在 `Properties/Resources.en-US.resx` 增加同名 key 与英文值。
3. 在 UI 中使用 `Loc` 或 `I18n.T / I18n.Tf` 引用 key。
